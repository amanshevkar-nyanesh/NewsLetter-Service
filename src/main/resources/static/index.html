📬 Newsletter Service API

Welcome to the Newsletter Service REST API

Base URL

http://localhost:8080/api/v1

API Endpoints

GET /api/v1/topics - Get all topics
POST /api/v1/topics - Create a topic
GET /api/v1/topics/{id} - Get topic by ID
GET /api/v1/subscribers - Get all subscribers
POST /api/v1/subscribers - Create a subscriber
POST /api/v1/subscribers/{id}/subscribe/{topicId} - Subscribe to topic
GET /api/v1/contents - Get all contents
POST /api/v1/contents - Create content
GET /api/v1/contents/topic/{topicId} - Get contents by topic

Health Check

GET /actuator/health - Application health status
